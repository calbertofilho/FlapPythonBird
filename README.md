# FlapPythonBird
### _Clone do jogo *FlappyBird* escrito em *Python*_<br />utilizando a biblioteca *Pygame*

Dependências:
* Biblioteca pygame
~~~sh
python -m pip install -U pygame
~~~ 

Requisitos: 
* Clonar o repositório:
~~~sh
python3 -m venv .env
source .env/bin/activate
pip3 install -r requirements.txt
~~~
