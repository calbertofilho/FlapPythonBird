# FlapPythonBird
Clone do jogo FlappyBird, desenvolvido em Python, utilizando a biblioteca Pygame


Dependências:
- Biblioteca pygame
~~~sh
python -m pip install -U pygame
~~~ 

Requisitos para clonar o repositório:
~~~sh
python3 -m venv .env
source .env/bin/activate
pip3 install -r requirements.txt
~~~
